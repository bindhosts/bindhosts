## Часто задаваемые вопросы

### Реклама не заблокирована в Chrome

- Не забудьте отключить umount для Chrome.
- **KernelSU:**
  1. Суперпользователь
  2. Chrome
  3. Пользовательский
  4. отключить размонтирование модулей<br><img src="https://raw.githubusercontent.com/bindhosts/bindhosts/master/Documentation/screenshots/umount_chrome.png" 
          onerror="this.onerror=null;this.src='https://raw.gitmirror.com/bindhosts/bindhosts/master/Documentation/screenshots/umount_chrome.png';"
          width="100%" alt="Disable umount modules screenshot">
- Это также применимо к другим браузерам на основе Chrome и Chromium.
- Распространенная проблема с настройками KernelSU по умолчанию.
