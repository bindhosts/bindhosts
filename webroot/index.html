          Blacklist Sources

          • Blacklist sources for action.sh, comment out to enable other sources
          • Example:
            127.0.0.1 host to block
            0.0.0.0 host.to.block

          • Alternatively you can also add links to custom rulsets here, mainly used to circumvent dns based filering by upstream (ISP, telecom, network provider), they wont be filter out.
          • Example:
            https://someonewhocares.org/hosts/zero/hosts

            https://pgl.yoyo.org/adservers/serverlist.php?hostformat=hosts&showintro=0&mimetype=plaintext

            https://raw.githubusercontent.com/StevenBlack/hosts/refs/heads/master/hosts

            Warning! Huge list:
            https://raw.githubusercontent.com/jerryn70/GoodbyeAds/master/Hosts/GoodbyeAds.txt
